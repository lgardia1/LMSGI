# **4.-Citas:**

Las citas se generar utilizando el carácter mayor que **>** al comienzo del bloque de texto.

# **5.-Listas**

Para crear listas desordenadas utiliza * asteriscos, - guiones, o + símbolo de suma.

* Elemento de lista 1
- ELemento de lista 2
+ Elemento de lista 3 
- Elemento de lista 4
* Elemento de lista 5
+ Elemento de lista 6

Guiones , asteriscos y el símbolo de suma se ven indiferentes.
Para listas anidadas se añade cuatro espacios despues del siguiente.
 * Elemento de lista 1
     * Elemento de lista 2


# **6.-Listas ordenadas**
Para crear listas ordenadas debes utilizar la sintaxis de tipo: «número.» **1.**. Al igual que ocurre con las listas desordenadas, también podrás anidarlas o combinarlas.
1. Elemento de lista
2. Elemtno de lista
   * Elemento de lista


# **7.-Links o enlaces en línea**

Añadir enlaces a una publicación, más que común, hoy en día es algo casi obligatorio. Con Markdown tendrás varias formas de hacerlo.
"[enlace en línea]([http://www.limni.net](https://www.wikipedia.org/))"
